# 模考模板列表 — 給 Gemini 檢查

模考 Type 2「文脈規定」（填空題）跟 Type 4「用法」（單字用法題）的句子是用「模板 + 隨機抽詞」生成的。模板本身的好壞決定題目品質 — 如果模板過於 specific（例如「毎日___を飲みます」需要可喝的東西），隨機抽到「警官 / 学校」就會變廢話。

請對每個模板判斷：
- **OK** — 對該詞性下絕大多數單字都能造合理句子
- **太 specific** — 只對某語意子分類合理（飲料/地點/時間等），建議改通用或刪除
- **語法怪** — 接續格式有問題

對「太 specific / 語法怪」請給出**替換建議**或建議刪除。

## Type 2: 文脈規定模板 (`CONTEXT_TEMPLATES`)

填空題格式：「___」是空格、要填詞。filter 限定可填的詞性。

### N5
| 模板 | filter（接受的詞性） |
|---|---|
| `毎日＿＿を飲みます。` | 名 |
| `＿＿に友達と会います。` | 名 |
| `＿＿で日本語を勉強します。` | 名 |
| `天気が＿＿です。` | い形 / な形 |
| `この本は＿＿です。` | い形 / な形 |

### N4
| 模板 | filter |
|---|---|
| `試験に＿＿ように、毎日勉強している。` | 動 |
| `先生に＿＿と言われた。` | 動 |
| `＿＿のに、とても高い。` | い形 / な形 |
| `＿＿ために、早起きしました。` | 名 / 動 |

### N3
| 模板 | filter |
|---|---|
| `彼は＿＿として知られている。` | 名 |
| `この問題＿＿、調査を行った。` | 名 |
| `経験＿＿、この仕事は難しい。` | 名 |
| `天気が＿＿ので、出かけましょう。` | い形 / な形 |

### N2
| 模板 | filter |
|---|---|
| `環境問題は＿＿にわたって議論されてきた。` | 名 |
| `＿＿に関する報告書を提出した。` | 名 |
| `彼女は努力した＿＿、合格した。` | 名 |

### N1
| 模板 | filter |
|---|---|
| `この法案は長年の議論＿＿、成立した。` | 名 |
| `経済危機を＿＿として、社会が変化した。` | 名 |

## Type 4: 用法題模板（`makeUsageSentence`）

「{w}」替換成測試的單字。每題給 4 選項：1 個正確套用 + 3 個其他詞硬塞同模板（製造錯誤用法），所以模板要對該詞性下**大多數**單字合理。

| 詞性 | 模板列表 |
|---|---|
| **名（名詞）** | 1. `{w}はとても大切です。`<br>2. `昨日{w}を見ました。`<br>3. `{w}が好きです。`<br>4. `{w}について話しました。` |
| **動（動詞）** | 1. `{w}ことが大好きです。`<br>2. `毎日{w}ています。`<br>3. `もっと{w}たいです。` |
| **い形** | 1. `この部屋は{w}です。`<br>2. `とても{w}料理でした。`<br>3. `{w}天気ですね。` |
| **な形** | 1. `彼女は{w}な人です。`<br>2. `この町は{w}です。`<br>3. `{w}な場所に行きたい。` |
| **副** | 1. `{w}勉強しました。`<br>2. `{w}歩きます。` |

## 回覆格式建議

```
Type 2:
- 「毎日＿＿を飲みます。」: 太 specific（要可喝的詞）→ 建議改成「＿＿について話しました。」
- 「天気が＿＿です。」: OK
- ...

Type 4:
- 名詞模板 3「{w}が好きです。」: OK
- 動詞模板 1「{w}ことが大好きです。」: 語法怪（需要動詞辞書形，多數動詞 OK 但動作性動詞「歩く」之類 awkward）
- ...
```
