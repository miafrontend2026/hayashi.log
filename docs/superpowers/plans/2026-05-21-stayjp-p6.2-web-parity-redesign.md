# StayJP P6.2 — Web-Parity Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps use `- [ ]`.

> **Supersedes P6.1.** P6.1 added daily mode + categorized sections but kept indigo + flat row design. P6.2 fully restyles to match `stayjp.study` actual deployment as observed via browser inspection 2026-05-21.

**Goal:** Make StayJP App's 單字 + 文法 + 今日 tabs visually and structurally match the web's deployed look:
- Warm terracotta palette (`#D4654A` primary, `#4A7FD4` secondary blue, `#FAF9F6` warm cream background)
- Each word/grammar item is its **own card with breathing room** — no dense flat rows
- 3-card stack at top of each browse tab: 打卡熱力圖 → 總進度 → 今日批次(含吉祥物說明)
- Daily mode (default ON) shows just today's 20-word / 10-point batch
- Category section headers with colored letter badge + name + count + thin underline
- Kana rendered in secondary blue (`#4A7FD4`) — a signature touch we missed
- Mascot illustration placeholder (a `<View>` shape now; real asset later)

**Architecture changes from P6/P6.1:**
- New design tokens in `src/theme.ts` — exported colors, used inline in screens (avoiding NativeWind class explosion)
- AeroCard retired or repurposed; new `StayCard` matches web's `--bg2 + --bd + --sh` look (warm whites + soft shadow)
- Word row becomes `WordCard` (its own card, mb-3)
- 3 hero cards: `StreakCalendar`, `LevelProgressCard`, `TodayBatchCard`
- Indigo classes (`bg-indigo-*`, `text-indigo-*`, `border-indigo-*`) sweep → terracotta tokens

```tsx
// src/vocab/WordCard.tsx
import { memo } from "react";
import { Pressable, Text, View } from "react-native";
import { Ionicons } from "@expo/vector-icons";
import { StayCard } from "../components/StayCard";
import { colors } from "../theme";
import { playWord } from "../lib/audio";
import { srsDotColor } from "./srsDotColor";
import { srsKey } from "../srs/algo";
import type { SrsData } from "../srs/types";
import type { Word, Level } from "../types/content";

interface Props {
  word: Word;
  level: Level;
  srs: SrsData;
  isFav: boolean;
  onPress: () => void;
  onToggleFav: () => void;
}

const CAT_LABEL: Record<string, string> = {
  "名": "名詞", "動": "動詞", "い形": "い形", "な形": "な形", "副": "副詞", "他": "其他",
};

export const WordCard = memo(function WordCard({ word, level, srs, isFav, onPress, onToggleFav }: Props) {
  const entry = srs[srsKey(level, word.w)];
  const dot = srsDotColor(entry);

  return (
    <Pressable onPress={onPress} style={{ marginBottom: 10 }}>
      <StayCard className="px-5 py-4">
        <View className="flex-row items-start">
          {/* dot */}
          <View style={{ width: 8, height: 8, borderRadius: 4, backgroundColor: dot, marginTop: 10, marginRight: 12 }} />
          {/* main column: kanji / kana / meaning */}
          <View className="flex-1 min-w-0">
            <Text style={{ fontFamily: "NotoSansJP_500Medium", color: colors.text, fontSize: 22, fontWeight: "600" }}>
              {word.w}
            </Text>
            {word.r && word.r !== word.w ? (
              <Text style={{ fontFamily: "NotoSansJP_400Regular", color: colors.secondary, fontSize: 14, marginTop: 2 }}>
                {word.r}
              </Text>
            ) : null}
            {word.m ? (
              <Text style={{ color: colors.textMuted, fontSize: 14, marginTop: 6 }}>
                {word.m}
              </Text>
            ) : null}
          </View>
          {/* right cluster: speaker / cat badge / star */}
          <View className="flex-row items-center gap-2 ml-2">
            <Pressable onPress={() => void playWord(word.r || word.w)} hitSlop={8}>
              <Ionicons name="volume-medium" size={20} color={colors.secondary} />
            </Pressable>
            <View style={{ backgroundColor: colors.primary, paddingHorizontal: 8, paddingVertical: 2, borderRadius: 999 }}>
              <Text style={{ color: "white", fontSize: 11, fontWeight: "600" }}>{CAT_LABEL[word.c] ?? word.c}</Text>
            </View>
            <Pressable onPress={onToggleFav} hitSlop={8}>
              <Ionicons name={isFav ? "star" : "star-outline"} size={18} color={isFav ? colors.primary : colors.textSubtle} />
            </Pressable>
          </View>
        </View>
      </StayCard>
    </Pressable>
  );
});
```
